# parkease-frontend
Parkease frontend

## Project Description
This is a parking management system for tracking vehicle entry and exit.  The system will support
accurate record keeping, automated fee calculation, receipt generation, and daily revenue
reporting.

## Pages
index.html: For attendant/admin login and granting secure access to the system.

dashboard.hmtl: For overview of the system activities, key information  and  quick access to main features

register.html: Form for vehicle entry and registration.

signout.html: For calculating fees and generating receipts

reports.html: for admin/attendant view for daily revenue and transcation history

## How to run
1.Clone the repo:git clone https://github.com/DianaNamirembe/parkease-frontend
2.Open `index.html` in your browser (no server needed)
3.Login with: username `admin` password `admin123`

## ## Features Implemented
- [x] Login page with role-based redirect
- [x] Attendant Dashboard
- [x] Vehicle Registration with form validation
- [x] Sign-out page with fee calculation
- [x] Admin Reports page with table filtering
